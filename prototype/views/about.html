About this prototype

This prototype application is a small proof of concept of the OpenTrials project. We've taken a single data source - a database of schizophrenia trial data from Cochrane - and build out the most essential aspects of the interface to display this data.

Behind the scenes, the process has involved some initial exploratory work in defining a data model for the project, and more specifically, the processing of the raw data source into normalised CSV and SQL formats for reuse.
